데모 쇼핑몰 어플리케이션

이 프로젝트는 **데모 쇼핑물 애플리케이션**의 핵심 구성 요소인 게시판의 데이터 무결성과 사용자 인증(비밀번호) 기능을 시연합니다.

게시판 바로가기 새 글 작성

게시판의 주요 특징

보안 처리된 글쓰기

모든 게시글은 비밀번호를 설정하여, 작성자 외 무단 수정/삭제를 방지합니다.

이미지 첨부 지원

게시글과 함께 이미지를 업로드하고 미리보기를 통해 첨부 파일 기능을 시연합니다.

조회수 카운트

글 상세 페이지 접속 시 조회수가 증가하는 기본적인 게시판 로직을 구현합니다.
